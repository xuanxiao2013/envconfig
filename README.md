## 环境配置

### 安装 zsh

````shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
````

### 安装 nvm

````shell
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
````

### 授权

````shell
chmod +x install.sh
````

### 安装
````shell
sudo ./install.sh
````
